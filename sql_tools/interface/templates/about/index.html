What the interface is for?

The SQL-Tools web interface is designed for developers who want to increase the simplicity of the use of library and its functions. This is a web interface that provides all the tools SQL Tools that you need for your project in a simple, secure & graphical way.


What can't I do this with tool?

Simply, nothing. The interface is packed with all the robust tools for every kind of work from simple data exploration to data science. It includes all the tools that are required by a developer to perform operations on data efficiently.


How it works?

It is a visual representation of all of the database operations that you may come across. It uses the SQL-Tools API behind the scenes to perform operations that you can also perform on your own by using the functions.
